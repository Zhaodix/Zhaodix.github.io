个人博客

  1. git
    • 在全局有你的name 和email
    • git config –list
    • git config -global user.name “”
    • git config -global user.email “”
    • 需要一个公钥
    • 在命令行执行 ssh-keygen -t rsa -C ‘上面的邮箱’ 一路回车就行
    • C/user（用户）/联想/.ssh/id_rsa.pub
    • 把里面的文件全部复制
    • 打开github
      • settings > SSH and GPG -> new SSH key
      • title 文本框空置
      • key 粘贴刚才复制的东西
  2. node
  3. npm

搭建博客

  1. 创建一个远程仓库，名称固定 “你的用户名.github.io”
    • 点击SSH 复制连接
  2. 本地命令行
    • 执行 npm i -g hexo-cli hexo
    • 本地创建一个文件夹
      • cd 进入文件夹
      • 打开 git bash
        • 执行 hexo init
        • 执行 npm install
        • 执行 hexo server 或者 hexo s
        • 可以在本地开启一个服务
        • 访问 localhost:4000
        • 看到本地个人博客页面
        • 执行 hexo g
        • 执行 hexo d
          • 如果执行 hexo d 的时候报错
          • 就执行 npm i hexo-deployer-git
          • 然后重复 hexo g 和 hexo d
        • 这个时候远程仓库刷新就有文件了
      • 远程仓库 点击 setting
        • 向下翻 会有一个网址
        • 网址就是 https://你的用户名.github.io/
  3. 使用hexo
    • 在本地文件夹中打开 git bash
      • 执行 hexo new 文件名
      • 在根目录下/source/posts/文件名.md
      • 在里面写内容
      • 编辑完成以后
        • 执行 hexo g
        • 执行 hexo d
      • 网址刷新就有了（注意缓存）
